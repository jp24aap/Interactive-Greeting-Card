# Interactive Celebration Card Template 🎉

A dynamic, front-end web application designed as a customizable digital greeting card. It features a date-locked entry gate, multimedia integration, and complex CSS/Canvas animations.

## 🔐 How to Test the Project
To unlock the main experience and trigger the animations, enter the following placeholder date on the landing screen:
**Secret Date:** `01` / `01` / `2025`

## 🛠️ Technical Features
* **Authentication Gate:** JavaScript logic that validates user input against a hardcoded date before revealing the primary DOM elements.
* **Canvas Animations:** Custom-built firework and particle explosion effects utilizing the HTML5 `<canvas>` API and `requestAnimationFrame` for smooth rendering.
* **Multimedia Integration:** Autoplaying background video overlays and synchronized audio elements.
* **Dynamic DOM Manipulation:** JavaScript intervals driving falling placeholder frames and floating emojis to create an immersive, continuous background effect.
* **Responsive UI:** CSS media queries ensuring fluid layouts and readable typography across mobile, tablet, and desktop viewports.

## 💻 Tech Stack
* HTML5
* CSS3 (Flexbox, Keyframe Animations, Backdrop Filters)
* Vanilla JavaScript (ES6+)
